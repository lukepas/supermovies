
Welcome to the SuperMovies NestJS backend repository! This README provides an overview of the project, its functionalities, and how to get started.

### Project Overview

SuperMovies is a NestJS backend application that provides essential functionalities for user authentication and email services. It includes features such as user registration, login, and email services for registration confirmation and password reminders.

### Functionalities

1. **Register** :

* Endpoint: `POST /auth/register`
* Description: Creates a new user account.
* Request Body: Requires `email` and `password`.

1. **Login** :

* Endpoint: `POST /auth/login`
* Description: Logs in an existing user.
* Request Body: Requires `email` and `password`.
* Response: Returns a JWT token upon successful login.

1. **Email Service** :

* Description: Provides email functionalities for registration confirmation and password reminders.
